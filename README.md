# 🌸 Iris Flower Classification

A production-style Machine Learning project built using Python, Pandas, Scikit-learn and Streamlit.

## Features

- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- Random Forest Classifier
- Model Evaluation
- Feature Importance
- Prediction Module
- Streamlit Web App

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit

## Run

```bash
python src/train.py
```

```bash
streamlit run app.py
```
